# Privacy Policy -- PR0TA Codex Plugin

**Last updated:** April 29, 2026

## Overview

The PR0TA Codex Plugin provides local skills and reference documentation that help Codex work with PR0TA creative production workflows.

## Data Collection

This plugin does not independently collect, store, or transmit personal data. It is a local instruction and documentation package for Codex.

## API Interactions

When you ask Codex to use PR0TA, Codex may make requests to PR0TA APIs using credentials you provide, such as a PR0TA Personal Access Token. Do not commit tokens, OAuth credentials, `.env` files, or project secrets to this repository.

API interactions with PR0TA are subject to PR0TA's own terms and privacy policy.

## Third-Party Services

This plugin can guide Codex workflows that interact with:

- PR0TA, for creative asset generation, project data, review rooms, and timeline workflows.
- Generation providers used by PR0TA, through PR0TA-managed API calls.
- GitHub, only when users clone, inspect, or update this plugin repository.

## Knowledge Files

The plugin includes PR0TA workflow and API reference files. These files should not contain personal data or credentials.
